##  О проекте

**Chatbot for Places** — это интеллектуальный помощник для поиска мест (кафе, рестораны, парки, музеи, развлечения), который анализирует **эмоциональное состояние пользователя** и предлагает места, соответствующие не только критериям (расположение, цена, кухня), но и **эмоциональным потребностям**.

###  Ключевая идея

> *Не просто «где поесть?», а «где мне будет хорошо с моим текущим настроением?»*

###  Возможности

- **Анализ эмоций** — определяет 28 эмоций из текста пользователя (радость, грусть, злость, удивление, романтика...)
- **Копинг-стратегии** — понимает, как пользователь хочет справиться с эмоцией (расслабиться, выплеснуть энергию, отвлечься...)
- **Семантический поиск** — находит места по смыслу, даже если теги не совпадают дословно
- **Гибридные рекомендации** — комбинирует структурированные данные и анализ отзывов
- **Байесовский рейтинг** — честно учитывает популярность мест (малоизвестные, но хорошие места не теряются)
